###1. twitter_pull.py
**_Twython:_** The scripts that pull from the Twitter API require the [Twython library](https://github.com/ryanmcgrath/twython) in order to run.
- **Purpose:** This is script is used to make request to the Twitter API for each of the candidates listed in the candidates_twitter.csv. It will then attempt to pull their twitter timeline and write their last 200 tweet's data (tweet, date, favorites, retweets, retweeted, and source) to tweets.csv.
